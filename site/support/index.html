Pocket Memory Pet

Support

Pocket Memory Pet is a local-first mobile app. No login or server account is required for the MVP.

Before Reporting

  • Confirm the app version and build number from the store listing or test build.
  • Include device model, operating system version, and the screen where the issue occurs.
  • Avoid sharing private diary text or personal information in public reports.

Support Channel

Use the public GitHub issue tracker for support requests: github.com/SeongWon1123/LAB/issues.

Privacy

Read the Privacy Policy for details about local-only MVP data.
