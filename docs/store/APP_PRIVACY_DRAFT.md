# App Store App Privacy Draft

## Data Linked To The User

None for the MVP.

## Data Used To Track The User

None.

The MVP does not include advertising SDKs, tracking SDKs, cross-app tracking, or third-party analytics.

## Data Not Linked To The User

None transmitted off-device.

## On-Device Only Data

The app stores the following locally:

- Pet name.
- Pet status values.
- Care history.
- Diary entries.
- Settings preferences.

This data remains on the device and is not collected by the developer in the MVP.

## Optional Permissions

- Notifications: Used only for local reminders if the user opts in.

The app does not request:

- Location.
- Contacts.
- Photos.
- Camera.
- Microphone.
- Bluetooth.
- Health data.

## Review Notes

No account is required. If notification permission is denied, the app remains fully usable.

- Confirm final binary still has no tracking, analytics, ads, or network-based data collection.
