# Design System

## Direction

Warm, soft, and tactile. The app should feel like a small pastel desk toy, not a modern game dashboard.

## Palette

- Primary Pink: `#F5AFC6`
- Cloud White: `#FFF4EC`
- Lavender Button: `#A78BD8`
- LCD Yellow: `#D9D08A`
- LCD Pixel Dark: `#32312B`
- Retro Brown: `#7A4E2D`
- Desk Brown: `#9B6A43`
- Dust Gray: `#6E6A61`
- Star Peach: `#FFC6A5`
- Rainbow Blue: `#95BFEA`
- Mint Accent: `#BFE8D4`
- Shadow Mauve: `#8B6F82`

## Screens

- Base design size: 390x844.
- Minimum target: 360x800.
- Portrait first.
- SafeArea required.

## Components

- `RetroDeviceShell`
- `LCDScreen`
- `PixelPet`
- `PixelGauge`
- `ThreeButtonPad`
- `DiaryCard`
- `SettingsToggle`
- `RetroDialog`

## Avoid

- Brand-confusing silhouettes.
- Existing character or icon copies.
- Hard neon palettes.
- Dense HUDs.
- Photoreal 3D toy renderings.
